# How the score is built

Every card shows a **score** from 0 to 100. It is a composite ranking, built from three parts and then clamped to the 0–100 range:

```
score = clamp(0, 100, state_score + liquidity_score + location_score)
```

<Callout tone="info">
The score is a ranking, **not a win probability**. A 96 is not a "96% chance." It means: among today's candidates, this one combines a strong state, deep liquidity, and clean location.
</Callout>

## The three parts

- **State score** — the base, set by the setup state. NARROW runs 82–94 (best), WIDE SNAPBACK 60, TRENDING 50, WATCH LOOSE 45, TOO TIGHT 30, MIDDLE 20.
- **Liquidity score** — `min(20, log10(avg_dollar_volume) × 2)`. It rewards dollar volume on a log scale and is **capped at +20**, so megacaps and merely-liquid names land close together.
- **Location score** — `+10` when price is clearly `above_cluster` or `below_cluster`, otherwise `+5`. Clean separation from the MA cluster is worth a small bonus.

## Using it

Sort and triage by score, but never trade the number alone. A high score from a NARROW name above the cluster is a real setup; a mid-score MIDDLE name is just liquid. Always confirm the state, the Prior45 zone, and the opening trigger before acting.
